# Credit Risk and Loan Portfolio Analytics

## Project Overview

This project analyzes a financial loan portfolio using Oracle SQL and Power BI.

The objective is to monitor loan performance, identify credit risk patterns, evaluate charged-off loans, and provide actionable insights through interactive dashboards.

## Tools & Technologies

- Oracle SQL
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dashboard Pages

### 1. Summary Dashboard
- Loan Applications
- Funded Amount
- Amount Received
- Interest Rate
- Debt-to-Income Ratio (DTI)

### 2. Portfolio Overview
- Monthly Trends
- Regional Analysis
- Loan Purpose Analysis
- Employee Length Analysis
- Home Ownership Analysis

### 3. Loan Details
- Transaction-Level Analysis
- Dynamic Filtering
- Detailed Loan Information

### 4. Credit Risk Analysis
- Charged-Off Loan Rate
- Charged-Off Funded Amount
- Risk by Grade
- Risk by Loan Purpose
- Risk by Home Ownership
- Top Risk States
- Risk Heatmap Matrix

## Key Insights

- Grade G loans show the highest default risk.
- Small Business loans present elevated charged-off rates.
- Certain states demonstrate significantly higher credit risk.
- Borrower characteristics influence repayment performance.
